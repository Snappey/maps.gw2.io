Created because I like scrolling around on maps when I'm wasting time and there were none for Guild Wars 2 that I enjoyed using. Little things like (hopefully) smooth performance when panning around but still being able to see all the relevant information and schedules for the currently active events for each expansion.

Marker Interaction

Clicking a marker to copy its chat link to your clipboard or double clicking to be taken to the wiki. Along with increased detail on major cities such as Lion's Arch and Arborstone.

Direct Links

You can also link others directly to a location on the map, look here is the Isle of Istan or using a chat link you add it to the end e.g. https://maps.gw2.io/tyria/[&BO8AAAA=] and a green circle will highlight it.

Drawing

Draw in the same style as in-game by holding the right click, your line will slowly fade after you let go.

Live Map (Experimental)

There is experimental support for a "Live Map" (requires an API key), allowing you to see yourself, guild, everyone or a special mix of all the above as markers on the map including in WvW. This feature is experimental and not officially supported. For support in getting set up can be found on the Gw2 Developer Discord below, with some detailed instructions provided by Xinterp.
